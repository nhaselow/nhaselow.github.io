Matrix Calculator
Spring 2016 - Fall 2017

Who wants to invert a matrix by hand? Multiply a couple of 4x4 matrices? Not me!

Built With:  Java,  GridBagLayout

What I Learned: This project made me appreciate the difficulty in making a seamless GUI, not only in the code but in the design. This respect has motivated my future course selections and will definitely influence how I approach future projects on the front-end!

About the Project: In parallel with a Linear Algebra course in the Spring of 2016, I developed a matrix calculator to help me do things that I didn't want to do! This application allows users to calculate basic matrix functions, including addition, multiplication, and transposition, along with more complicated ones such as inversions and determinant calculations. These operations can be done on matrices of any size, and operations can be combined via a custom expression reader.
