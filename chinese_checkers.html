Chinese Checkers
Fall 2015 - Spring 2016

Early in my Data Structures course, I struggled understanding a couple of key concepts (including generic types, graphs, and recursion), so I figured I'd make myself use them in a project!

Built With: Java

What I Learned: I learned the difficulties of going through a project from start to finish! Not only does the data manipulation have to function, but the UI must be easy to use and without flaws as well.

About the Project: This is an application that allows the user to play Chinese Checkers against other human players or computer players powered by a greedy move selection algorithm. The board is represented by a hexagonal lattice data structure with cube-coordinates to manage vertices and their edges, which not only connects nodes but gives direction between them and their neighbors.

Chinese Checkers was my first reasonably-sized completed project, and the first project that I really felt proud to see functioning and progressing!
